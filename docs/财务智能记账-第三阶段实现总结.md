## 三、历史数据学习

### 学习维度

| 维度 | 学习内容 | 应用场景 |
|-----|---------|---------|
| **科目偏好** | 企业常用科目、使用频率、平均金额 | 科目推荐排序 |
| **科目组合** | 经常一起使用的借贷科目对 | 凭证自动补全 |
| **辅助核算** | 常用辅助核算类型和值 | 辅助核算自动填充 |
| **摘要模式** | 各科目高频关键词 | 智能摘要补全 |
| **金额特征** | 各科目的金额范围、平均值 | 金额合理性检查 |

### 核心算法

```typescript
// 学习流程
1. 获取历史凭证数据（默认90天）
2. 科目偏好统计 - 按使用频率排序
3. 科目组合分析 - 同凭证中经常成对出现的科目
4. 摘要关键词提取 - 词频统计，提取前10个高频词
5. 金额分布分析 - 计算平均值、最小值、最大值
6. 保存学习模式 - 计算置信度，存入学习记录表
```

### 学习置信度计算

```typescript
// 科目偏好
confidence = min(0.95, 0.5 + usageCount * 0.02)

// 科目组合
confidence = min(0.95, 0.5 + pairCount * 0.05)

// 摘要模式
confidence = min(0.95, 0.5 + matchCount * 0.15 + baseConfidence * 0.3)

// 金额模式
confidence = max(0.3, 0.9 - deviation)
```

## 四、异常检测预警

### 预警类型

| 预警类型 | 严重程度 | 检测条件 | 处理建议 |
|---------|---------|---------|---------|
| **借贷不平衡** | high | 借方≠贷方 | 检查凭证分录 |
| **大额交易** | medium/high | 金额>阈值(默认1万) | 确认真实性，保留合同 |
| **疑似重复** | medium | 同一天相同金额 | 检查是否重复录入 |
| **现金超限** | medium | 现金交易>1000元 | 建议银行转账 |
| **税务风险** | medium | 招待费/福利费 | 注意扣除限额 |
| **跨期发票** | medium | 开票时间>1年 | 及时报销处理 |
| **无附件** | low | 大额无影像 | 上传发票扫描件 |

### 检测流程

```
凭证创建/修改
    ↓
触发异常检测
    ↓
┌─────────────────┬─────────────────┬─────────────────┐
│ 借贷平衡检查    │ 大额交易检查    │ 重复凭证检查    │
│ 现金限额检查    │ 税务风险检查    │                 │
└─────────────────┴─────────────────┴─────────────────┘
    ↓
发现异常 → 创建预警记录 → 发送通知 → 待处理队列
    ↓
用户处理 → 解决/忽略 → 更新状态
```

### 预警统计

  high: 2,         // 高危
  medium: 5,       // 中危
  low: 3           // 低危
}

## 五、智能推荐优化

### 1. 智能科目推荐

**推荐维度**：
- 摘要关键词匹配（权重40%）
- 交易对手历史（权重30%）
- 金额范围匹配（权重20%）
- 业务类型映射（权重10%）

**推荐示例**：
```typescript
[
  {
    accountId: 123,
    accountCode: '660201',
    accountName: '管理费用-办公费',
    confidence: 0.85,
    reason: '摘要匹配"办公用品、打印纸"关键词',
    basedOn: 'summary_pattern'
  },
  {
    accountId: 124,
    accountCode: '660202',
    accountName: '管理费用-邮电费',
    confidence: 0.65,
    reason: '历史上与"顺丰快递"经常使用此科目',
    basedOn: 'counterparty_history'
  }
]
```

### 2. 智能摘要补全

**功能**：输入部分摘要，自动推荐完整摘要

**示例**：
- 输入："5月"
- 推荐："5月份办公用品采购"（置信度0.9）
- 备选：["5月房租", "5月工资", "5月差旅费"]

### 3. 智能凭证补全

**功能**：根据已有分录，推荐对方科目完成借贷平衡

**场景**：
```typescript
// 用户已录入：
借：管理费用-办公费 1000

// 系统推荐补全：
贷：银行存款 1000 （置信度0.9）
// 理由：历史上"管理费用-办公费"常与"银行存款"配对使用
```

### 4. 智能模板推荐

**推荐逻辑**：
- 摘要匹配度（40%）
- 科目重叠度（40%）
- 使用频率（20%）

## 七、财务智能仪表板

### 综合数据视图

```typescript
GET /finance/intelligence/dashboard

{
  // 预警统计
  alerts: {
    total: 100,
    active: 10,
    high: 2,
    medium: 5,
    low: 3
  },
  
  // 推荐统计
  recommendations: {
    total: 500,
    accepted: 350,
    acceptanceRate: 0.70,  // 70%采纳率
    byType: {
      account: { total: 200, accepted: 140 },
      template: { total: 150, accepted: 120 },
      summary: { total: 150, accepted: 90 }
    }
  },
  
  // 学习统计
  learning: {
    totalPatterns: 150  // 发现150个记账模式
  },
  
  // 健康评分
  health: {
    score: 85,          // 财务健康分（0-100）
    status: 'healthy'  // healthy/warning/critical
  }
}
```

### 健康分计算逻辑

```typescript
// 基础分100
let score = 100;

// 扣除异常分
score -= highAlerts * 10;      // 高危预警-10分
score -= mediumAlerts * 5;      // 中危预警-5分
score -= lowAlerts * 2;         // 低危预警-2分

// 加分项
score += acceptanceRate * 10;   // 推荐采纳率加分

// 限制范围
score = Math.max(0, Math.min(100, Math.round(score)));
```
