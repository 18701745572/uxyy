| CMBC | 民生银行 | CSV, Excel |
| CITIC | 中信银行 | CSV, Excel |
| PAB | 平安银行 | CSV, Excel |
| HXB | 华夏银行 | CSV, Excel |
| ALIPAY | 支付宝 | CSV |
| WXPAY | 微信支付 | CSV |

### 智能匹配规则

```typescript
// 匹配逻辑
1. 交易类型识别（工资/税款/转账）
2. 备注关键词识别（货款/费用/报销）
3. 交易对手匹配（客户/供应商表）
4. 金额特征识别（大额/小额）
5. 默认分类（收入/支出）

// 匹配结果
- unmatched: 无法匹配
- matched: 高置信度匹配（>0.7）
- voucher_created: 已生成凭证
```

### 核心API

```typescript
// 解析CSV
POST /finance/extension/bank/import
{
  "bankCode": "ICBC",
  "bankAccount": "6222021234567890",
  "csvContent": "...CSV内容...",
  "hasHeader": true
}

// 获取流水列表
GET /finance/extension/bank/statements?matchStatus=unmatched&page=1

// 智能匹配
POST /finance/extension/bank/statements/{id}/match

// 生成凭证
POST /finance/extension/bank/statements/{id}/generate-voucher
{
  "accountId": 123  // 可选，自定义科目
}

// 批量匹配
POST /finance/extension/bank/batch-match
```
