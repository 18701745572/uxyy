### 2. 企业/项目类别

**推荐填写**：**人工智能应用/智能体系统**

**详细版本**（如表格空间允许）：
```
人工智能 > 智能体应用 > 企业数字化经营智能体平台
```

**类别选择理由**：

| 政策匹配 | 说明 |
|----------|------|
| 智能体专项补贴 | 直接匹配"智能体"类别，最高400万 |
| 算力券/模型券 | AI企业标配，各100万 |
| 场景开发补贴 | AI+企业数字化场景，最高1000万 |
| 种子基金 | AI赛道优先投资 |
| 高新企业认定 | 人工智能领域通过率更高 |

**避免使用的类别**：
- ❌ "软件开发"（过于普通，无法体现AI属性）
- ❌ "信息服务"（政策匹配度低）
- ❌ "电子商务"（与AI关联度不高）

### 3. 项目简述

**精简版**（50字以内）：
```
基于大语言模型的智能体平台，为小微企业提供AI驱动的客户管理、进销存、CRM、财务一体化智能经营解决方案。
```

**标准版**（100字以内）：
```
优效营是国内首个专注小微企业经营的智能体平台，基于大语言模型和AI Agent技术，为零售、批发、服务业小微企业提供智能销售预测、客户流失预警、AI话术推荐、智能采购建议等自主决策能力，实现"AI员工"辅助经营，让小微企业享受AI技术红利。
```

**完整版**（200字以内）：
```
【项目背景】
中国小微企业超5000万家，数字化程度低，经营决策依赖个人经验，急需AI技术赋能。

【解决方案】
优效营打造"小微企业经营智能体平台"，基于大语言模型构建AI Agent系统，提供：
1. 智能销售预测：预测未来7-30天销售额，自动生成采购建议
2. 客户智能管理：流失预警+AI话术推荐，提升转化率
3. 财务智能分析：AI纠错+健康度诊断，降低财务风险
4. 经营决策辅助：自然语言问答，数据驱动决策

【技术亮点】
- 垂直领域大模型微调
- 多智能体协作架构
- 低代码场景配置

【社会价值】
让小微企业以低成本享受AI技术，预计帮助10万+企业实现智能化经营升级。
```

**关键词植入**（便于政策审核）：
- ✅ 智能体/AI Agent
- ✅ 大语言模型
- ✅ 小微企业
- ✅ 数字化转型
- ✅ 自主决策
- ✅ AI赋能

**推荐填写**：

```
【投资规模】首期投资500-1000万元，计划3年累计投资3000万元
【团队规模】首期团队15-20人，3年扩展至50人
【营收预期】首年营收目标500万元，3年突破5000万元
【用户规模】首年服务1000家企业，3年服务10万家企业
【算力需求】年度算力投入约200万元（已申请算力券）
【场地需求】初期200-300㎡办公空间，3年扩展至800㎡
```

**规模设计理由**：

| 指标 | 设定值 | 政策匹配 |
|------|--------|----------|
| 投资500-1000万 | 中等规模 | 匹配种子基金投资区间 |
| 团队15-20人 | 初创型 | 符合小镇初创企业定位 |
| 首年500万营收 | 可达成 | 体现商业化能力 |
| 算力200万/年 | 较高 | 支撑算力券申请（100万） |
| 场地200-300㎡ | 适中 | 享受办公空间补贴 |
